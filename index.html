Divra Bloxd Userscript

Divra whose full name is Divra Bloxd Userscript is a userscript that allows you to change how bloxd looks
and feels to be better than any userscript that has or will ever exist. Name one user script and the team will
match it!

bloxmaHow it works

Divra works by pasting the code in a user script manager preferably Tampermonkey and just going into bloxd should automatically work. You don't need to know any Javascript to use this user script because it's already coded for you.

bloxmaWhy Divra

Divra may be a new Userscript but updates are coming in fast! There is no excuse to try Divra right away! You can delete it in the user script manager dashboard if you don't like it! Divra is also very simple and easy to use. If you need any help, just simply join our discord community; they are very kind and helpful.

How to install it

It isn't hard to install and use it! Here are the steps.

  1. Install a userscript manager
  2. Go to this website
  3. Click Install This Script

Keybord Shortcuts

  • J : Stop Timer
  • K : Start Timer
  • L : Hide / Show Divra Buttons
  • ; : Hide / Show Divra Keystrokes
  • ` : Hide / Show bloxd.io UI
